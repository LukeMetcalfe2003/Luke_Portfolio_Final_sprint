Project 1: Snake Game

This is a simple snake game that I created using Python. The objective of the game is to eat the food that appears on the screen and grow the snake. If the snake collides with itself or the walls, the game is over. The game keeps track of the player's score and displays it on the screen.




Project 2: Gary Blue's Diner
